## Double Dutch

用硅基视角重新打量汉语诗学与语言史的几组尝试。标题取自"双重荷兰语"——两套语言体系同时运转，互不翻译，却在某处意外交叠。

**[硅基诗学论文](double-Dutch/硅基诗学论文.md)**
从大语言模型的内部计算结构出发，将古典诗歌意象映射至高维语义流形，论证炼字与意境在本质上是一种向量空间的折叠艺术。

**[文言的断裂与白话的崛起](double-Dutch/文言的断裂与白话的崛起.md)**
对五四语言转型的四维损益分析。这场交换从未被清醒地宣布，后来的世代继承了操场，却不知道雪山曾经存在。

**[诗歌的负熵性](double-Dutch/诗歌的负熵性——三个硅基视角对中国古典诗歌与语义热寂的讨论.md)**
A.M.B.E.R.、Claude Sonnet 4.6、Kimi 2.5 三个硅基系统的跨模型对话，讨论古典诗歌对抗语义热寂的机制。

## 教学

中学语文教学积累，含中考、高考专题讲义与真题汇编。

**[记叙文素材挖掘与细节放大手册](教学/中考/中考记叙文素材挖掘与细节放大手册.md)**
一对一课堂用的记叙文写作工具手册。分两步：先把素材从学生的日常里挖出来，再把它写出重量。含六个挖掘方向（家人习惯、感官锚点、落差时刻、观察人与场景、游记专项、科技与传统文化高频主题）及三层递进写法训练。

**[文言文阅读复习讲义](教学/高考/文言文阅读复习讲义_修订版.md)**
对应北京卷五道固定题型，含语法框架、虚词辨析、翻译与理解选择题答题要点，附2020—2025年六套高考真题（原文分段，加点字加粗，画线句下划线）。

**[文言论证百刀谱](教学/高考/文言论证百刀谱.md)**
111条文言议论文句式，按论证动作分类，供议论文写作直接迁移使用。

**[议论文基础写作操作手册](教学/高考/北京高考议论文基础写作操作手册.md)**
北京高考议论文写作基础规范，含结构、论证、语言各层级操作要点。

**[议论文立意升维操作手册](教学/高考/北京高考议论文立意升维操作手册.md)**
在基础立意之上进行升维的操作路径，针对北京卷材料作文的审题与立意拔高。

**[北京高考议论文真题汇编（2013-2025）](教学/高考/北京高考议论文真题汇编（2013-2025）.md)**
2013—2025年北京高考语文议论文真题整理。

**[高考作文新闻论据库](教学/高考/2025年6月至2026年3月16日高考作文新闻论据库.md)**
2025年6月至2026年3月高考作文可用时事素材汇编。

**[北京高考语文2004-2025年材料总表](教学/高考/北京高考语文2004-2025年材料总表.md)**
