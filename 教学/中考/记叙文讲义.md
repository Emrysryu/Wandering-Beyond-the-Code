# 中考记叙文复习讲义
## 一、散文基础知识
### 1. 散文与小说的区别
| 体裁 | 核心特征 | 构成要素 |
|-----|-----|-----|
| 小说 | 虚构 | 人物、情节、环境 |
| 散文 | 真实 | 线索、主旨、情感、感悟 |
| 传记 | 真实 | 人物、事件 |

### 2. 散文的线索类型
线索是作者将材料串联起来的"红线"或"寄托物"。线索在文中的体现，多半在标题、开头、结尾和过渡段的段首、段尾等处。
|-----|-----|-----|
｜时间线索｜如：泰戈尔《金色花》以时间的推移为叙事线索｜
｜空间线索｜如：鲁迅《从百草园到三味书屋》《藤野先生》以时空变换为线索｜
｜人物线索｜如：朱自清《背影》以父亲的背影为线索｜
｜事物线索｜如：茅盾《白杨礼赞》以"不平凡"的白杨树为物线，以"不平凡"为情线｜
｜事件线索｜如：《散步》以"一件小事"为线索｜
｜情感线索｜如：《雨的四季》以对雨的爱恋或对自然对生活的热爱为抒情线索｜
### 3. 散文的结构形式
｜结构形式｜行文思路｜
|-----|-----|-----|
｜静赏式｜景点概述 → 景点静赏 → 景点联想｜
｜参游式｜进入景点 → 依次欣赏 → 赏景联想 → 离开景点｜
｜象征式｜物的描述 → 物的性格 → 由物及人 → 表达志向｜
｜怀念式｜睹眼前景（人、事） → 思从前景（人、事） → 抒怀念情｜
｜叙史式｜一语通篇 → 追述史实 → 抚今而感慨抒情｜
**4. 散文常见的语言风格**
•	豪放：题材多为雄心伟业；景象境界开阔；动词富有力度；抒情激越昂扬；多用排比、夸张、反复、反问等
•	柔婉：对象纤巧细致，情感细致缠绵；画面色调柔和；少用排比、夸张、反问等
•	直露：表达作者的感受和观点比较直接
•	含蓄：托物言志、借景抒情；多用象征、设问、比喻等
•	质朴(平实)：语言通俗化、口语化；少用修辞手法和描绘性语言
•	华丽(典雅)：讲究节奏韵律；多用描绘性语言、引用、排比、对偶、用典等
•	庄重：话题严肃，语言凝重；句式整齐、完整而绵长
•	诙谐：语气轻松幽默；多用夸张、反语、拟人等
•	轻松：以闲谈式的生活语言入文，生活化、口语化
•	简洁：语势流畅，多用短句
•	细腻：多用长句，多用对比和辩证性语言
