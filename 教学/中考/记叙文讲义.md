# 中考记叙文复习讲义

## 一、散文基础知识

### 1. 散文与小说的区别

| 体裁 | 核心特征 | 构成要素 |
|-----|-----|-----|
| 小说 | 虚构 | 人物、情节、环境 |
| 散文 | 真实 | 线索、主旨、情感、感悟 |
| 传记 | 真实 | 人物、事件 |

---

### 2. 散文的线索类型

**线索**是作者将材料串联起来的“红线”或“寄托物”。  
线索在文章中的体现，多见于：

- 标题  
- 开头  
- 结尾  
- 过渡段的段首或段尾  

| 线索类型 | 示例 |
|-----|-----|
| 时间线索 | 泰戈尔《金色花》以时间的推移为叙事线索 |
| 空间线索 | 鲁迅《从百草园到三味书屋》《藤野先生》以时空变化为线索 |
| 人物线索 | 朱自清《背影》以“父亲的背影”为线索 |
| 事物线索 | 茅盾《白杨礼赞》以“不平凡的白杨树”为物线 |
| 事件线索 | 《散步》以“一件小事”为线索 |
| 情感线索 | 《雨的四季》以对雨、对自然的热爱为抒情线索 |

---

### 3. 散文的结构形式

| 结构形式 | 行文思路 |
|-----|-----|
| 静赏式 | 景点概述 → 景点静赏 → 景点联想 |
| 参游式 | 进入景点 → 依次欣赏 → 赏景联想 → 离开景点 |
| 象征式 | 物的描述 → 物的性格 → 由物及人 → 表达志向 |
| 怀念式 | 睹眼前景 → 思从前景 → 抒怀念情 |
| 叙史式 | 一语通篇 → 追述史实 → 抚今而抒情 |

---

### 4. 散文常见的语言风格

- **豪放**  
  题材多为雄心伟业；景象境界开阔；动词富有力度；抒情激越昂扬；多用排比、夸张、反复、反问等。

- **柔婉**  
  对象纤巧细致，情感细腻缠绵；画面色调柔和；较少使用排比、夸张、反问等。

- **直露**  
  表达作者的感受和观点较为直接。

- **含蓄**  
  常用托物言志、借景抒情；多使用象征、设问、比喻等手法。

- **质朴（平实）**  
  语言通俗、口语化；较少使用复杂修辞。

- **华丽（典雅）**  
  讲究节奏与韵律；多用描写性语言、排比、对偶、引用和用典。

- **庄重**  
  话题严肃，语言凝重；句式整齐、完整而绵长。

- **诙谐**  
  语气轻松幽默；多用夸张、反语、拟人等手法。

- **轻松**  
  以生活化、闲谈式语言入文，表达自然亲切。

- **简洁**  
  语势流畅，多使用短句，表达干净利落。

- **细腻**  
  多用长句，善于对比与细节描写，情感表达丰富。
